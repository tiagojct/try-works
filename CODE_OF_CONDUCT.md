# Code of Conduct

This project follows the Contributor Covenant, version 2.1. In short: be
respectful, assume good faith, and keep criticism about the work, not the
person. Unacceptable behaviour includes harassment, personal attacks, and
discriminatory language.

Report concerns to the maintainer at the address on the project homepage.
Maintainers may remove contributions and contributors that violate this code.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
